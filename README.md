# SmartKeyTurkey

Private development workspace for the SmartKeyTurkey multilingual corporate website.

## Scope

- Custom WordPress theme optimized for Elementor Pro
- Site-owned plugin for structured content and business functionality
- English source content with Persian, Arabic, and Turkish via WPML
- Light and dark themes using `#84c341` as the primary brand color
- B2C property discovery for Istanbul, Ankara, Izmir, and Antalya
- B2B petrochemical catalog and qualified inquiry workflow
- Attractions and sourced editorial coverage of Turkish laws, news, and events

## Repository policy

This repository contains custom source code, configuration examples, documentation, and safe export artifacts only. Passwords, API keys, license keys, production configuration, database dumps, uploads, and personal data must never be committed.

## Delivery workflow

Work is developed and verified on staging before production. Each meaningful change receives a focused commit. Daily Slack updates record completed work, evidence, blockers, and the next task.

See [docs/ROADMAP.md](docs/ROADMAP.md) and [docs/PROJECT-CHARTER.md](docs/PROJECT-CHARTER.md).
