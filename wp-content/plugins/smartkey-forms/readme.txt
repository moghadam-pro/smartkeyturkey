Independent form building, private submission storage and notification integration for SmartKeyTurkey.

== Description ==

SmartKey Forms separates forms and submissions from SmartKey Core. It provides:

* Native WordPress form records.
* A structured field-definition builder.
* An Elementor-compatible shortcode.
* Private database-only submissions.
* Contact Form 7 RFQ capture compatibility.
* Notification events for Telegram and future providers.

No email is sent for managed SmartKeyTurkey forms.
