# Turkey attractions editorial model

## Purpose

The `skt_attraction` content type supports English-first destination discovery without mixing time-sensitive visitor guidance into property or news content. Every record has a city, visitor snapshot, official source URL and last-reviewed date.

## Phase-one fields

| Field | Purpose |
| --- | --- |
| Title and excerpt | Search/result-card identity and concise summary |
| Main content | Original SmartKeyTurkey editorial overview and planning guidance |
| Attraction city | Istanbul, Ankara, Izmir or Antalya archive relationship |
| District / area | Practical location context without publishing a street address where unnecessary |
| Attraction type | Heritage, archaeological, memorial, district or future controlled vocabulary |
| Suggested duration | Non-binding planning estimate |
| Best for | Audience and experience fit |
| Coordinates | Google Maps destination link |
| Official source URL | Primary factual reference |
| Last reviewed | Freshness and editorial-review signal |

## Source and claims policy

- Phase-one factual summaries use official GoTürkiye destination material.
- Copy is rewritten for SmartKeyTurkey; it is not a reproduction of the source page.
- Opening hours, ticket prices, worship access and temporary closures are not hard-coded because they change frequently.
- Each guide tells visitors to confirm current arrangements with the official destination before travelling.
- AI-authored content remains subject to later human editorial review.
