# Elementor petrochemical template specification

Status: implementation ready  
Last updated: 2026-07-21

## Design goals

- Make technical product discovery fast for B2B buyers.
- Keep SmartKey's representative role visible and unambiguous.
- Prioritize qualified RFQ conversion without implying live stock or fixed pricing.
- Use semantic headings, crawlable text and stable internal links.
- Support system-driven light and dark themes with `#84c341` as the primary accent.

## Archive template

Elementor Theme Builder draft: `SmartKey — Petrochemical Product Archive` (`337`)

1. Breadcrumbs
2. Intro section
   - H1: `Petrochemical Products & Polymer Grades`
   - Short value proposition matching the archive meta description
   - Primary CTA: `Request a Quote`
3. Product-family navigation
   - Product families shown as crawlable links
   - Current family visually identified on taxonomy archives
4. Product results
   - Three-column desktop, two-column tablet and one-column mobile grid
   - Featured image with descriptive alt text
   - Product family, product title and concise excerpt
   - CTA label: `View specifications`
5. Pagination
   - Crawlable previous/next navigation
6. B2B trust and RFQ section
   - Representative disclosure
   - Availability and specifications confirmed per inquiry

SEO requirements:

- Exactly one H1 generated by the archive template.
- Product card titles use H2 or H3 consistently.
- Do not duplicate the Rank Math archive description with large blocks of repetitive copy.
- Filters must not create indexable duplicate query URLs in phase one.
- Maintain pagination links and do not replace them with JavaScript-only infinite scroll.

## Single-product template

Elementor Theme Builder draft: `SmartKey — Petrochemical Single Product` (`339`)

1. Breadcrumbs using Product Family as the primary taxonomy
2. Product hero
   - Featured image
   - H1 dynamic post title
   - Product family and grade
   - Availability label: `Confirm by RFQ`
   - Primary CTA: `Request current terms`
3. Post content
   - Original product overview
   - Applications
   - Technical properties table
4. Commercial inquiry panel
   - Product/grade prefill target for the Petrochemical RFQ form
   - Quantity, packaging, Incoterm, destination and document request prompts
5. Representative disclosure
6. Related products from the same Product Family

SEO requirements:

- Exactly one H1.
- Technical property labels remain HTML text, not image-only content.
- Include contextual links to the family archive and RFQ journey.
- Use the featured image as the primary social image when appropriate.
- Do not output manufacturer, offer, price, stock or certification schema unless supported by reviewed data.

Use CSS `prefers-color-scheme` as the initial automatic theme signal. A future manual selector must preserve the user's explicit choice.

## Responsive and accessibility acceptance

- No horizontal overflow at 320 px.
- Touch targets are at least 44 × 44 px.
- Visible keyboard focus is retained on links, buttons and form controls.
- Text and interactive colors meet WCAG AA contrast.
- Technical tables receive a horizontally scrollable wrapper on small screens.
- Images have explicit dimensions or aspect ratios to minimize layout shift.
- CTA text describes the action and avoids generic labels such as `Click here`.
