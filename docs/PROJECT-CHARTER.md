# Project charter

## Business outcome

Rebuild SmartKeyTurkey as a credible multilingual corporate platform that generates qualified B2C property leads and B2B petrochemical inquiries while also growing organic reach through destination and editorial content.

## Confirmed decisions

- Start date: 2026-07-19
- Primary language: English
- Additional languages: Persian, Arabic, Turkish
- Translation system: WPML
- Page builder: Elementor Pro
- Primary color: `#84c341`
- Fonts: Inter, Roboto, Vazir
- Launch property cities: Istanbul, Ankara, Izmir, Antalya
- Initial petrochemical catalog: approximately 100 products
- SmartKeyTurkey works directly with properties and projects under its control; current documents, availability and transaction terms are supplied on request
- Initial content and translations may be AI-assisted and will be flagged for later human review
- Conversion priorities will be re-ranked using post-development testing data

## Initial conversion set

- Property lead form
- Petrochemical request-for-quotation form
- Consultation request
- Phone click
- WhatsApp click

## Governance

The site manager reviews and approves work progressively. Legal, financial, property availability, pricing, ownership, product specifications, and regulatory claims require explicit review before being presented as verified facts.

## Security rules

- Never place passwords, tokens, license keys, database exports, personal data, or production configuration in GitHub or Slack.
- Use a secure credential-sharing route with revocable, least-privilege accounts.
- Verify backup and rollback before production changes.
- Perform implementation and content migration on staging first.

## Current dependencies

- The owner-provided list of approximately 100 petrochemical products and available grade/document data is still required for catalog import.
- The RFQ recipient mailbox must be confirmed before end-to-end delivery testing.
- Contact details, social links and additional brand imagery will be supplied progressively by the owner.
